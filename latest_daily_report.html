通用 Survey / NPS 竞品
AskNicely NPS / AI Agent 6月2日发布后复盘
Insights Agent + Response Agent 是本周最重要信号：竞品已经把客户反馈处理从 dashboard 推向 proactive workflow。
Insightto 可先做简化版：每周自动邮件 + 负面反馈提醒 + 建议动作。
Survicate Research Hub Changelog
Research Hub 把反馈数据组织成研究项目，并强调 direct quotes 和可追溯证据。Insightto 的 AI 结论也必须附证据，否则商家难以信任。
对小店主来说不需要完整研究项目，但需要“问题—证据—动作”的报告结构。
Shopify 原生竞品 Zigpoll / Fairing / Grapevine Shopify App Store / 官方博客
Zigpoll 覆盖多触发调查，Fairing 绑定广告归因，Grapevine 用低价无限响应抢基础 NPS。三者共同挤压 Insightto 的基础问卷价值。
Insightto 必须把定位提升到“AI customer insight for Shopify retention”，而不是“another survey app”。
